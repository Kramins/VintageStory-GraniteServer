# GraniteServer


GraniteServer is an alpha-stage server management and web API project designed to work with the [Kramins/container-vintagestory-server](https://github.com/Kramins/container-vintagestory-server) container for Vintage Story servers.

**Note:** The container is not yet published to Docker Hub. Please refer to the GitHub repository for build and usage instructions.
